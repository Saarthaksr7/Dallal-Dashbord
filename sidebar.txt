6. Terminal
    - Path: /ssh
    - Icon: Terminal
    - Submenus:
        - **SSH Console:** (/ssh/console) - Secure, browser-based terminal emulator with session logging and host selection dropdown.
        - **Command History:** (/ssh/history) - View of recently executed remote commands with execution time, user, and host details.
        - **SFTP Browser:** (/ssh/sftp) - Integrated file browser allowing secure file transfer and basic directory navigation using stored SSH credentials.
    - Status: **POPULATED** (Implementation requires Paramiko/GoTTY backend for connection and secure key management).

7. RDP
    - Path: /rdp
    - Icon: Monitor
    - Submenus:
        - **RDP Sessions:** (/rdp/sessions) - Interface to launch new RDP connections using stored credentials and view active sessions.
        - **Screen Recordings:** (/rdp/recordings) - (Optional Admin Feature) Storage and playback of recorded remote sessions for audit purposes.
    - Status: **POPULATED** (Implementation requires a secure web-based RDP gateway integration like Apache Guacamole).
